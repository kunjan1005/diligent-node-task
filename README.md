# node task

## Getting started
 
 These instuctions will help you to set up and run the project on your local machine

 ### Prerequisites
   -Node.js (version 20.11.0)
   -npm (version 10.2.4)
   -Database (posgresql)

### Installing
    npm i
    npm run start:dev
